# Digital Nurture 5.0 - Java FSE React Hands-On Solutions

## Overview

This repository contains my solutions for the **Digital Nurture 5.0 - Java Full Stack Engineer (FSE) React Deep Skilling Program**.

The program is designed to strengthen software engineering fundamentals and full-stack development skills through a combination of self-learning, hands-on exercises, and practical implementation.

This repository includes the **mandatory hands-on exercises** completed as part of the 7-week learning journey.

---

## Program Duration

- Duration: 7 Weeks
- Track: Java FSE React
- Learning Type: Deep Skilling
- Assessment: Weekly Hands-ons + Final Assessment
