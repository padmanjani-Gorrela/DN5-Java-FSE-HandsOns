# Exercise 1 - Control Structures

## Objective

Learn PL/SQL control structures using loops, conditions, and cursors.

---

## Scenarios

### Scenario 1

Apply a 1% discount to loan interest rates for customers above 60 years.

### Scenario 2

Promote customers to VIP status if their balance exceeds ₹10,000.

### Scenario 3

Generate reminders for loans due within the next 30 days.
