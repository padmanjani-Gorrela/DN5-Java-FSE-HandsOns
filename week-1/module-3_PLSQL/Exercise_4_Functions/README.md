# Exercise 4 - Functions

## Objective

Learn how to create and use PL/SQL Functions that return values.

---

## Scenarios

### Scenario 1

Calculate customer age from Date of Birth.

### Scenario 2

Calculate monthly loan installment.

### Scenario 3

Check whether an account has sufficient balance.

---

## Files

- CalculateAge.sql
- CalculateMonthlyInstallment.sql
- HasSufficientBalance.sql
- output.txt

---

## Concepts Used

- Functions
- RETURN Statement
- IF-ELSE
- MONTHS_BETWEEN()
- ROUND()
- Exception Handling

---

## Workflow

Input Parameters

↓

Function

↓

Processing

↓

Return Result

↓

Display Output

---

## Advantages

- Reusable code
- Returns a single value
- Better modularity
- Improved maintainability
