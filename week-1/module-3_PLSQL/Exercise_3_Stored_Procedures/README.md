# Exercise 3 - Stored Procedures

## Objective

Learn to create reusable PL/SQL stored procedures for performing business operations.

---

## Scenarios

### Scenario 1

Process monthly interest for all savings accounts.

### Scenario 2

Update employee bonus based on department.

### Scenario 3

Transfer funds between customer accounts.

---

## Files

- ProcessMonthlyInterest.sql
- UpdateEmployeeBonus.sql
- TransferFunds.sql
- output.txt

---

## Concepts Used

- Stored Procedures
- Parameters
- UPDATE Statement
- COMMIT
- ROLLBACK
- Exception Handling
- RAISE_APPLICATION_ERROR

---

## Workflow

Client

↓

Stored Procedure

↓

Database Operation

↓

Commit / Rollback

↓

Display Output

---

## Advantages

- Code Reusability
- Better Performance
- Centralized Business Logic
- Improved Security
